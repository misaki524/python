PROJECT 01

家計簿アプリ

支出・収入・予算・借金・バイト勤怠までを一元管理する、スマホ／PC両対応のSPA。 Google スプレッドシートをデータベースとして使い、サーバー費用ゼロで運用できる家計簿です。

Web App / SPA
主な機能
  • 月／週ごとの支出入力・一覧・合計、日平均の自動計算
  • 予算設定と消化率・残額の可視化
  • 収支分析グラフ（カテゴリ別・推移）と節約日カウント
  • 既存データのCSV一括取込（列マッピング自動検出付き）
  • 固定費カテゴリ管理・カテゴリの追加
  • PC＝サイドバー／スマホ＝タブバーの自動切替UI
技術的な工夫
  • サーバーレス構成：フロント（Vue）＋ Google Apps Script ＋ スプレッドシートで、運用コスト0円・インフラ管理不要を実現。
  • バッチ送信：CSV一括取込は20件ずつに分割して送信し、GASの実行時間制限を回避。
  • 秘密情報をフロントに持たない設計：接続先URLは各端末のlocalStorageに保持し、コードには含めない。
  • 関心の分離：API通信（services）／画面（views）／ユーティリティ（utils）を分離した見通しの良い構成。

PROJECT 02

ノベルゲーム「深夜のコンビニ」

霧に閉ざされた深夜のコンビニで、夜明けまで生き延びるホラー／サスペンスADV。 全8エンディングのマルチ分岐と周回要素を持ち、企画・シナリオ・実装をすべて担当しました。

Game / Ren'Py
主な機能・ボリューム
  • プロローグ＋全5章のマルチ分岐シナリオ（最大4分岐の大分岐あり）
  • 到達条件の異なる全8エンディング（トゥルー／隠しエンド含む）
  • 手がかり収集システム（Tabキーで一覧表示）
  • エンディングギャラリー・周回数に応じた解放要素
  • 時刻HUD、画面揺れ・フラッシュなどのホラー演出
設計の工夫
  • 再利用可能なエンジン層：演出・フラグ管理・HUD・セーブ等を engine/ モジュールに分離し、他作品へ流用可能に。
  • フラグの一元管理：全ストーリーフラグを辞書で定義し、開始時に一括リセット。手がかり数も配列から自動算出。
  • データ駆動のUI：エンディング一覧や手がかりリストをデータ配列から生成し、追加・変更を容易に。
  • 永続化と周回設計：到達エンドと周回数をpersistentで保存し、2周目以降の追加分岐を制御。

PROJECT 03

勤怠管理アプリ

出退勤・休憩・日報を一元管理するスマホ向けPWA。打刻するとセッションごとの勤務時間を自動計算し、 1日に複数回の出退勤にも対応した実用ツールです（v1.1.0）。

Web App / PWA
主な機能
  • 出勤・退勤打刻と、セッションごとの勤務時間の自動計算
  • 1日に複数回の出退勤を記録し、合計勤務時間を自動集計
  • 休憩開始／終了の記録と休憩時間の集計（複数回対応）
  • 箇条書き形式の日報作成・更新（あとから追記・編集可）
  • 勤務状態を色分け表示するステータスバッジ＋秒単位の時計
技術的な工夫
  • サーバーレス／無料運用：GAS＋スプレッドシートのみで完結し、Googleアカウントだけで動作。
  • セッション設計：1日複数回の出退勤を配列で管理し、合計勤務時間をサーバー側で算出（v1.1.0で対応）。
  • 状態整合性の担保：未退勤セッションの有無をチェックし、二重出勤や休憩中の退勤などの不整合を防止。
  • 安全なUX：退勤はアニメーション付きカスタムモーダルで確認し、通信中はボタンを無効化して二重送信を防止。
  • 仕様書ドリブン開発：機能IDごとに処理関数・事前条件・レスポンスを定義した仕様書を整備。
