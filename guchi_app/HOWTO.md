# 愚痴アプリ セットアップガイド

## 概要

Google Apps Script + Google Sheets + Gemini API で動作する個人用愚痴日記アプリです。
スマートフォンのホーム画面からPWAとして使えます。費用は無料です。

---

## 必要なもの

- Googleアカウント
- Gemini APIキー（無料）

---

## セットアップ手順

### 1. Google Sheets の準備

1. [Google Sheets](https://sheets.google.com) で新しいスプレッドシートを作成
2. シート名（左下タブ）を `Entries` に変更
3. 1行目に以下のヘッダーを入力（A1〜G1）

| A | B | C | D | E | F | G |
|---|---|---|---|---|---|---|
| entry_id | timestamp | date_str | emotion_tag | content | ai_response | mood_score |

4. URLからスプレッドシートIDをコピー
   - 例: `https://docs.google.com/spreadsheets/d/` **`ここがID`** `/edit`

---

### 2. Gemini API キーの取得

1. [Google AI Studio](https://aistudio.google.com/apikey) にアクセス
2. 「Create API key」をクリック
3. 生成されたキーをコピーして安全な場所に保存

---

### 3. Google Apps Script の設定

1. [Google Apps Script](https://script.google.com) にアクセス
2. 「新しいプロジェクト」をクリック
3. プロジェクト名を `愚痴アプリ` に変更

#### Code.gs の設定

1. デフォルトの `myFunction` の内容を全て削除
2. `Code.gs` の内容を貼り付け
3. 以下の2箇所を自分のIDとキーに書き換え

```javascript
const SPREADSHEET_ID = 'YOUR_SPREADSHEET_ID';  // ← ステップ1でコピーしたID
const GEMINI_API_KEY = 'YOUR_GEMINI_API_KEY';   // ← ステップ2でコピーしたキー
```

#### index.html の追加

1. 左メニューの「＋」→「HTMLファイル」を選択
2. ファイル名を `index`（拡張子なし）に設定
3. `index.html` の内容を貼り付け（既存の内容を全て置き換え）

---

### 4. Webアプリとしてデプロイ

1. 右上の「デプロイ」→「新しいデプロイ」をクリック
2. 「種類の選択」の⚙️ アイコン →「ウェブアプリ」を選択
3. 以下の設定にする

   | 項目 | 設定値 |
   |------|--------|
   | 説明 | 愚痴アプリ |
   | 次のユーザーとして実行 | 自分 |
   | アクセスできるユーザー | 自分のみ |

4. 「デプロイ」をクリック
5. アカウント選択画面が出たら自分のGoogleアカウントを選ぶ
6. **「このアプリはGoogleで確認されていません」と表示される（正常）**
   1. 左下の「詳細」をクリック
   2. 下部に表示される「愚痴アプリ（安全ではないページ）に移動」をクリック
   3. 「許可」をクリック

   > ⚠️ これは自分で作ったスクリプトなので安全です。  
   > Googleの審査を受けていないアプリには必ずこの警告が出ます。

7. 表示されたWebアプリのURLをコピー

---

### 5. スマホのホーム画面に追加（PWA化）

**iPhone（Safari）の場合**
1. SafariでWebアプリのURLを開く
2. 下部の共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」をタップ
4. 「追加」をタップ

**Androidの場合**
1. ChromeでWebアプリのURLを開く
2. 右上のメニュー（⋮）→「ホーム画面に追加」
3. 「追加」をタップ

---

## 画面の使い方

### ✏️ 書くタブ

1. 感情タグ（😡怒り / 😢悲しみ / 😤ストレス / 😴疲れ / 😰不安）を選ぶ
2. 愚痴を入力
3. 「AIに話しかける」ボタンをタップ
4. 1〜3秒後にAIからの返答が表示される

### 📖 日記タブ

- 過去の愚痴が新しい順に表示される
- 長い内容は「続きを読む」で展開できる
- 🗑️ ボタンで削除できる（確認ダイアログあり）
- 「もっと見る」で過去の記録を追加読み込み

### 📊 グラフタブ

- **気持ちのトレンド**: 期間内の感情変化を折れ線グラフで表示
  - Y軸が絵文字で各感情を示す
  - 各点の色はその日の最多感情の色
- **まとめ**: 合計件数、最多感情、最も多かった日
- **感情の内訳**: 各感情の割合をドーナツグラフで表示
- 期間切替ピル（7日 / 14日 / 30日）で表示範囲を変更

---

## 更新のたびにやること（コードを変更した場合）

1. GASエディタで修正を保存（Ctrl+S / Cmd+S）
2. 「デプロイ」→「デプロイを管理」
3. 鉛筆アイコン（編集）→バージョンを「新しいバージョン」に変更
4. 「デプロイ」をクリック
5. URLは同じなのでスマホ側での操作は不要

---

## トラブルシューティング

| 症状 | 原因 | 対処 |
|------|------|------|
| 送信しても何も起きない | SPREADSHEET_IDが間違っている | Code.gsのIDを確認 |
| AIの返答が表示されない | GEMINI_API_KEYが間違っている | キーを確認・再生成 |
| 「アクセス権限がない」エラー | デプロイ設定が「自分のみ」以外 | デプロイ設定を確認 |
| グラフが表示されない | まだデータが0件 | 愚痴を投稿してから確認 |
| スマホでPWAが開かない | URLを直接開いてキャッシュ更新 | ホーム画面から削除して再追加 |

## ファイル構成

```
guchi_app/
  Code.gs     ← GASバックエンド（編集不要、ID/キーの設定のみ）
  index.html  ← フロントエンド
  HOWTO.md    ← このファイル
```

## データの確認・バックアップ

- すべてのデータはGoogle Sheetsの`Entries`シートに保存されています
- 通常のスプレッドシートとして閲覧・エクスポートできます
- 削除したデータは復元できないのでご注意ください
