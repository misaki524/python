# Cursed Caverns（呪われた洞窟）」　説明

## 概要
このファイルはPyxelを使ったゲームプロジェクトのメインモジュールです。
`Game`クラスを実行することで、ゲームが開始されます。

## 構成
- `from game import Game`
    - 同じディレクトリ内の`game.py`から`Game`クラスをインポートします。
- `Game()`
    - `Game`クラスのインスタンスを生成し、ゲームを開始します。

## 使い方
1. この`main.py`を実行すると、`game.py`内の`Game`クラスが呼び出され、ゲームがスタートします。
2. ゲームのロジックや画面表示などの詳細は`game.py`に記述されています。

## 例
```bash
python main.py
```

## ゲーム内容

このゲームは「Cursed Caverns（呪われた洞窟）」を舞台にしたアクションゲームです。
プレイヤーは敵を避けたり倒したりしながら、洞窟内を進みます。
スコアが表示され、ゴールに到達するとゲームクリアとなります。

### ゲームの終了方法

- 敵やトラップなどでやられると「ゲームオーバー」となり終了します。
- ゴール地点に到達すると「ゲームクリア」となり終了します。
- ゲームを途中で終了したい場合は、ウィンドウを閉じてください。
