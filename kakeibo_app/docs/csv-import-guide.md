# 既存スプレッドシートからのCSV取り込み手順

## 概要

既存のExcel/スプレッドシートで管理していた家計データを、CSVファイル経由でアプリに一括取り込みする手順。

---

## 手順

### 1. 既存スプレッドシートからCSVエクスポート

1. Google スプレッドシートで既存の家計データを開く
2. **ファイル** → **ダウンロード** → **カンマ区切り形式(.csv)**
3. CSVファイルがダウンロードされる

※ Excelの場合: **ファイル** → **名前を付けて保存** → **CSV UTF-8(コンマ区切り)(.csv)** を選択

### 2. CSVファイルの確認

取込前に、CSVに以下の列があるか確認してください:

| 必須/任意 | フィールド | 説明 | 例 |
|-----------|----------|------|-----|
| **必須** | 日付 | 支出の日付 | 2026-01-15, 1/15, 2026/1/15 |
| **必須** | 金額 | 支出金額（数値） | 1500, ¥1,500 |
| 任意 | 品名/内容 | 支出の説明 | スーパーで買い物 |
| 任意 | カテゴリ | 支出カテゴリ | 食費, 日用品 |
| 任意 | 支払い方法 | 現金/カード | 現金, カード, cash, card |

**注意**: 日付と金額の列は必須です。

### 3. アプリでの取込操作

1. アプリの **「取込」** タブを開く
2. **「CSVファイルを選択」** でダウンロードしたCSVを選択
3. CSVの列名が自動検出される
   - 自動検出がうまくいかない場合は、各ドロップダウンで手動選択
4. **プレビュー** で先頭5件のデータを確認
5. 問題なければ **「○○件を取り込む」** ボタンをクリック
6. Google Sheetsのスプレッドシートにデータが書き込まれる

### 4. 取込後の確認

- **月の支出** タブで過去のデータが表示されるか確認
- 月切替で各月のデータが見えるか確認
- 合計金額が元のスプレッドシートと一致するか確認

---

## カテゴリのマッピング

アプリのカテゴリ一覧:

| カテゴリ | 用途 |
|---------|------|
| サブスク | Netflix, Spotify等の月額サービス |
| 家賃 | 家賃・管理費 |
| ローン | 各種ローン返済 |
| 病院 | 医療費・薬代 |
| 光熱費 | 電気・ガス・水道 |
| 携帯電話 | スマホ料金 |
| 自由金 | 娯楽・趣味 |
| 交通費 | 電車・バス・タクシー |
| 食費 | 食料品・外食 |
| 日用品 | 洗剤・生活用品 |
| ペット | ペット関連 |

既存スプシのカテゴリ名がこれと異なる場合:
- CSV内のカテゴリ列がそのまま取り込まれる
- 取込後にスプレッドシート上で直接カテゴリ名を修正可能

## トラブルシューティング

| 問題 | 対処 |
|------|------|
| CSVが文字化けする | UTF-8エンコードで保存し直す |
| 金額に¥やカンマが含まれる | アプリ側で自動的に数値変換を試みるが、うまくいかない場合はCSV側で数値のみに整形 |
| 日付フォーマットが認識されない | YYYY-MM-DD形式に統一するのが最も確実 |
| 「取込」ボタンが押せない | 設定画面でGoogle Sheets APIの設定と認証を完了する |
