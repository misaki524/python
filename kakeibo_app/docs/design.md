# 家計簿アプリ 設計書

## 1. アプリ概要

| 項目 | 内容 |
|------|------|
| アプリ名 | 家計簿アプリ |
| フロントエンド | Vue.js 3 + Vite |
| バックエンド/DB | Google Sheets API（スプレッドシートをDBとして使用） |
| グラフ | Chart.js |
| CSV取込 | PapaParse |
| 対象端末 | スマホ（メイン）、PC（サブ） |

## 3. 画面構成

| タブ | パス | 機能 |
|------|------|------|
| 月の支出 | /monthly | 当月の支出一覧・入力・合計 |
| 週の支出 | /weekly | 今週の支出一覧・合計・日平均 |
| 予算 | /budget | 予算設定・消化率・残額計算 |
| バイト | /work | 勤怠入力・月間集計 |
| 借金 | /debt | 借金一覧・返済管理 |
| 分析 | /analysis | グラフ・収支・節約日カウント |
| 取込 | /import | 既存スプシからCSV取込 |
| 設定 | /settings | Google Sheets API設定 |
